JstMStudio 应用程序

JstMStudio 应用程序是由 JstMStudio 制作的开源多用途跨平台应用程序。 JstMStudio 应用程序目前包含许多免费功能，如视频流、待办事项等等，而且没有任何隐藏费用。 JstMStudio App 完全免费，所有人都可以在我们的 Github 上查看我们的源代码。



使用 JstMStudio 的好处

开源

JstMStudio 项目是完全开源的，每个人都可以在此处在我们的 Github 组织中根据 AGPL 3.0 开源 copyleft 许可证查看、研究和使用我们的源代码。 用于构建JstMStudio网站和平台的一切都是完全开源的，大家可以免费查看源代码。 我们目前没有也没有计划开发闭源软件。

隐私友好

JstMStudio 还在我们所有的项目和网站中优先考虑我们的用户隐私，以便让用户可以安全地使用我们的功能和项目，而不会泄露他们的隐私而受到伤害。 我们从我们的项目中收集的用户数据也少于任何其他公司，我们不会共享或出售我们收集的任何数据，以高质量加密保护所有数据，并且不会创建关于我们每个人的个人资料 用户。

安全环境

在 JstMStudio 中，我们提供万维网 (WWW) 上第一个最安全的环境。 我们对平台和网站上的所有内容进行严格和严格的审核，以确保用户的安全。 我们还制定了严格的社区准则，我们对社区实施了该准则，以保持社区的安全和清洁。 我们所有的管理员和版主都是训练有素的精英管理员，以确保 JstMStudio 中的最佳环境。
