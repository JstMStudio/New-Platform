Contributing To JstMStudio

There are many stuff you could help and contribute to JstMStudio projects and help out the twam. Each and every of your help and contribution to our projects is appreciate by us no matter if it just a little typo or an error by the team.



Reporting Bugs

One way to contributing to the project without technical knowledge is to report bugs to the team. If you found a bugs on our website or software, please create a Github issue or email us and report it. It would help us a lot by improving our software so everyone could use it smoother.

If you could, you could also help us to fixed reported bugs when we not yet fixed the bugs. It would certainly help the team a lot through for saving times and efforts in finding bugs. After finish programming, you could submit a pull request for one of our volunteer to approve and deploy it too.



Adding content onto JstMStudio software

You could contribute to the project by adding some contents onto our software. You could submit sites or projects onto our search engine, adding videos, music and book onto our content library for everyone to enjoy and etc.



Codes

If you a developer, you could also help us to improve our software like improving our UI, helping developing and built our upcoming new features and project with the team and etc. You could also help and contribute to create more functions for our current services too.

Extension

JstMStudio Profile is the first ever social site that actually provide support to extension on users profile and posts. An extension is a software add-on that is installed on a program, enhancing its capabilities. Users could choose from a vary of extension to use in their profile and post in JstMStudio Profile extension library. JstMStudio Profile extension is built by official staff team and the community.

Users could install all plugins here into their code by adding it before the post elements end or on their profile. Please install the plugin by copying and pasting the code below.



Clap

Clap is one of the official extension of the JstMStudio Profile. It could be used in JstMStudio Profile users post. Users could clap to users post when this plugin installed.



Comment

Comment is one of the official extension of the JstMStudio Profile. Users could add them to their post for viewer to comment. Users could also use it to get feedback on their post too.



Like

Like is one of the official extension of the JstMStudio Profile. Users could add them to their post for viewer and writer to show their reaction to the post. Users could also react to show their support too.



Share

Share is one of the official extension of the JstMStudio Profile. Users could add them to their post for viewer to share their post onto social media. Users could also react to show their support too by sharing the post.



Updown

Updown is one of the official extension of the JstMStudio Profile. Users could add them to their post for viewer and writer to show their reaction to the post. Users could also react to show their support too. Updown could also used for polls too.

API

JstMStudio provided a lot of services and features inside our platform software, JstMStudio App Platform. JstMStudio API is an Application Programming Interface for JstMStudio App Platform that provide support for all developer to implement services like in JstMStudio App Platform into their website without developing the features themselves for totally free of charge with no fees. It is built for everyone to use and not for earning money from users like developer and website owner.

JstMStudio API is an open source API ( Application Programming Interface ). Everyone could view, read and study all of JstMStudio API source codes on our official Github repository. JstMStudio API is built using HTML, CSS and vanilla JavaScript unlike any traditional API too and licensed under AGPL 3.0.
